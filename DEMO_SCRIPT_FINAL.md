## HARD RULES

1. **DO NOT** start with "Hi, my name is X and our project is Y." Boring. Start with the story.
2. **DO NOT** show the login/registration flow. Nobody cares. Start already logged in.
3. **DO NOT** show loading screens. Pre-load everything or cut in editing.
4. **DO NOT** read text off the screen. The UI should be self-explanatory.
5. **DO NOT** list technologies one by one. Say "two AI models work together" — judges will read the README for details.
6. **DO** show the 3D scene for at least 30 seconds. It's your visual differentiator.
7. **DO** give a bad answer on purpose to show the AI catching mistakes. This is 10x more impressive than a perfect run.
8. **DO** show the word-by-word highlighting. It's visually striking.
9. **DO** mention InsForge and TinyFish BY NAME. Sponsors notice. Sponsors vote.
10. **DO** end with impact, not tech. "Interview coaching shouldn't cost $300" > "We used 6 APIs"
